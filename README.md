# GestaoSoftware API

API para gerenciamento de software, desenvolvida em .NET 8 com PostgreSQL.

---

## Descrição

GestaoSoftware é uma API para gerenciamento de softwares e seus processos, permitindo:

- Cadastro e gerenciamento de softwares
- Controle de usuários e permissões
- Autenticação via JWT
- Integração com PostgreSQL para persistência de dados

---

## Tecnologias

- .NET 8
- ASP.NET Core Web API
- Entity Framework Core
- PostgreSQL
- Docker e Docker Compose
- JWT Authentication
- Swagger

---

## Start projeto

GestaoSoftware docker compose up --build

Obs: Link disponível somente ao rodar o projeto
